If you want to run a specific list on a specific algorithm
Input 1 –
    	make insertion.o
	or
	make heapsort.o
	or
	make quicksort.o
Input 2 –
       a.out "name of file"
------------------------------------------------------------
If you want to run all algorithms with all text files
Input -
	make all
------------------------------------------------------------
If you want to run specific algorithm with all files
Input -
	make insertion
	or
	make heapsort
	or
	make quicksort
------------------------------------------------------------
If you want to run a certain set of data on each file
Input -
	make tenthousand
	or
	make twentythousand
	or 
	make fortythousand
	or
	make eightythousand
